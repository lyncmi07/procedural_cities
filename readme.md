### Running program
From command line:   
`java -jar terrain_gen.jar`

My apologies for the nasty camera controls. It wasn't a particular priority during this project.  

### Controls 
W: Forwards  
S: Backwards  
A: Left  
D: Right  
Z: Up  
X: Down  

Q: Rotate Yaw Left  
E: Rotate Yaw Right  
Up Arrow: Rotate Pitch Down  
Down Arrow: Rotate Pitch Up
